How to Create a Case Manager

Case Managers will log into this website to enter CSI and Student Visit records.
Because of this, Case Managers are not added to this system in the same way that Students, Caregivers, or Sponsors are.
Instead, they need to be added as a User of this system and granted the Case Manager role.

If you have the 'Manage Users' role, click on 'User List' in the menu on the left side of the page.
Then, click the 'Create New User' button in the top-left corner of the screen.
Enter the Case Manager's information, ensuring that you check the box next to Case Manager in the Roles section.
The Case Manager will then appear in the Case Managers list.
